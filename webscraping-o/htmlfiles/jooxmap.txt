         <h2 class="item-title">
          <a href="/2-uncategorised/9-contributing-to-xmap.html">
           About Xmap
          </a>
         </h2>
         <p>
          Xmap is a powerful site map generator for Joomla! CMS. Xmap allows you to create a map of your site using the structure of your menus. It not only generates a site map for the users, but it can also generate a XML sitemap to use with Google, Bing and other spiders/robots.
         </p>
         <p>
          Xmap creates the site map based on the menus of your site. You can add as many menus you want to the site map and arrange them as you like. Also, you can create any number of site maps with different options for each.
         </p>
         <p>
          Xmap supports a large number of components and this number is constantly growing.
         </p>
