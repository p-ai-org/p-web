     <div class="qa-main">
      <h1>
       <span class="highlight">
        Question2Answer
       </span>
       is a free and open source platform for Q&amp;A sites.
      </h1>
      <div class="twocolumn">
       <h2>
        What is a Question and Answer site?
       </h2>
       <p>
        A Q&amp;A site helps your online community to share knowledge. People with questions get the answers they need. The community is enriched by commenting, voting, notifications, points and rankings.
       </p>
       <h2>
        Why offer Q&amp;A on my site?
       </h2>
       <p>
        Your members will enjoy the interaction that Q&amp;A enables, and will visit your site more regularly. In addition, many web searches are questions, so Q&amp;A content will attract search engine traffic.
       </p>
       <h2>
        What is Question2Answer?
       </h2>
       <p>
        Question2Answer (Q2A) is a popular open source Q&amp;A platform for PHP/MySQL, currently running on
        <a href="sites.php">
         24,662 sites in 40 languages
        </a>
        .
       </p>
       <h2>
        How do I get Q2A?
       </h2>
       <p>
        <a href="question2answer-latest.zip">
         <b>
          Download Question2Answer 1.8.5
         </b>
        </a>
        , then read
        <a href="https://docs.question2answer.org/install/">
         how to install
        </a>
        . Version 1.8.5 was
        <a href="versions.php">
         released
        </a>
        on July 15th, 2020. Also
        <a href="https://github.com/q2a/question2answer">
         on GitHub
        </a>
        .
       </p>
       <h2>
        What does Q2A need?
       </h2>
       <ul>
        <li>
         <a href="http://www.php.net/">
          PHP
         </a>
         5.2 or later, with the
         <a href="https://en.wikipedia.org/wiki/MySQLi">
          MySQLi
         </a>
         extension.
        </li>
        <li>
         <a href="http://www.mysql.com/">
          MySQL
         </a>
         MySQL 5.0 or later.
        </li>
       </ul>
       <p>
        It's strongly recommended to use the latest version of PHP (7.x) and MySQL if possible as they bring many performance enhancements and each major release of Q2A typically drops support for an older version.
       </p>
       <h2>
        What's the license?
       </h2>
       <p>
        <span class="highlight">
         Question2Answer is open source
        </span>
        ,
        <a href="license.php">
         licensed under GPL v2+
        </a>
        .
       </p>
       <h2>
        How can I help?
       </h2>
       <p>
        Contribute
        <a href="https://github.com/q2a/question2answer">
         via Github
        </a>
        or create a
        <a href="https://docs.question2answer.org/themes/">
         theme
        </a>
        ,
        <a href="https://docs.question2answer.org/plugins/tutorial/">
         plugin
        </a>
        or
        <a href="https://docs.question2answer.org/translate/">
         translation
        </a>
        .
       </p>

      <div class="twocolumn">
       <h2>
        Easy to set up
       </h2>
       <ul>
        <li>
         Q2A takes 5 minutes (or less!) to
         <a href="https://docs.question2answer.org/install/">
          install
         </a>
         .
        </li>
        <li>
         Easy styling with
         <a href="https://docs.question2answer.org/themes/">
          CSS themes
         </a>
         .
        </li>
        <li>
         Supports
         <a href="https://docs.question2answer.org/translate/">
          translation
         </a>
         into any language.
        </li>
        <li>
         Custom sidebar, widgets, pages and links.
        </li>
        <li>
         SEO features such as
         <a href="http://sitemaps.org/">
          XML Sitemap
         </a>
         and neat URLs.
        </li>
       </ul>
       <h2>
        Fast and secure
       </h2>
       <ul>
        <li>
         PHP/MySQL
         <a href="https://docs.question2answer.org/install/optimize/">
          scalable
         </a>
         to millions of users and posts.
        </li>
        <li>
         <a href="https://docs.question2answer.org/install/security/">
          Safe from
         </a>
         XSS, CSRF and SQL injection attacks.
        </li>
        <li>
         Beat spam with captchas, moderation and/or flagging.
        </li>
       </ul>
       <h2>
        Core Q&amp;A features
       </h2>
       <ul>
        <li>
         Fast integrated search engine.
        </li>
        <li>
         Categories (up to 4 levels deep) and/or tagging.
        </li>
        <li>
         Voting, comments, follow-on and closed questions.
        </li>
        <li>
         Points-based reputation management.
        </li>
        <li>
         RSS, email notifications and personal news feeds.
        </li>
       </ul>
       <h2>
        Built-in user account management...
       </h2>
       <ul>
        <li>
         Create experts, editors, moderators and admins.
        </li>
        <li>
         User avatars (or
         <a href="https://en.gravatar.com/">
          Gravatar
         </a>
         ) and custom fields.
        </li>
        <li>
         Private messages and public wall posts.
        </li>
        <li>
         Log in via Facebook or others (using
         <a href="https://docs.question2answer.org/addons/#plugins">
          plugins
         </a>
         ).
        </li>
       </ul>
       <h2>
        ...or integrate with existing sites
       </h2>
       <ul>
        <li>
         Out-of-the-box
         <a href="https://docs.question2answer.org/install/wordpress/">
          WordPress
         </a>
         and Joomla integration.
        </li>
        <li>
         Custom
         <a href="https://docs.question2answer.org/install/single-sign-on/">
          single sign-on
         </a>
         support for other sites.
        </li>
        <li>
         Deep integration and customization via
         <a href="https://docs.question2answer.org/plugins/">
          plugins
         </a>
         .
        </li>
       </ul>
