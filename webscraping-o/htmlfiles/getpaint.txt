         <h5>
          <b>
           About
          </b>
          <br/>
          Paint.NET is image and photo editing software for PCs that run Windows. It
          <a href="features.html">
           features
          </a>
          an intuitive and innovative user interface with 
				support for layers, unlimited undo, special effects, and a wide variety of 
				useful and powerful tools. An active and growing
          <a href="https://www.getpaint.net/redirect/forum.html">
           online 
				community
          </a>
          provides friendly help,
          <a href="https://www.getpaint.net/redirect/tutorials.html">
           tutorials
          </a>
          , and
          <a href="https://www.getpaint.net/redirect/plugins.html">
           plugins
          </a>
          .
         </h5>
         <h5>
          It started development as an undergraduate college senior 
				design project mentored by Microsoft, and is now maintained and developed 
				by Rick Brewster. 
				Originally intended as a free replacement for the
          <a href="https://en.wikipedia.org/wiki/Ms_Paint" target="_blank">
           Microsoft Paint
          </a>
          software that comes with Windows, it has grown into a powerful 
				yet simple image and photo editor tool. It has been compared to 
				other digital photo editing software packages such as
          <a href="https://en.wikipedia.org/wiki/Photoshop" target="_blank">
           Adobe
           <sup>
            <font size="1">
             ®
            </font>
           </sup>
           Photoshop
           <sup>
            <font size="1">
             ®
            </font>
           </sup>
          </a>
          , 
				Corel
          <sup>
           <font size="1">
            ®
           </font>
          </sup>
          <a href="https://en.wikipedia.org/wiki/Paint_Shop_Pro" target="_blank">
           Paint Shop Pro
           <sup>
            <font size="1">
             ®
            </font>
           </sup>
          </a>
          , 
				Microsoft Photo Editor, and
          <a href="https://en.wikipedia.org/wiki/GIMP" target="_blank">
           The GIMP
          </a>
          .
         </h5>
