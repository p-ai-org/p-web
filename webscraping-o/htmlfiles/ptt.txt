      <h2>
       What is Ptt?
      </h2>
      <p>
       批踢踢 (Ptt)  是以學術性質為目的，提供各專業學生實習的平台，而以電子佈告欄系統
     (BBS, Bulletin Board System)
    為主的一系列服務。期許在網際網路上建立起一個快速、即時、平等、免費，開放且自由的言論空間。批踢踢實業坊同時承諾
       <u>
        永久
       </u>
       學術中立，絕不商業化、絕不營利。
       <small>
        (詳細的權利與義務以及隱私權保障請參閱
        <a href="index.ua.html">
         使用者條款
        </a>
        )
       </small>
       <br/>
       目前在 Ptt/Ptt2 註冊的人數超過一百萬人，尖峰時段兩站容納超過十五萬名使用者
       <b>
        同時
       </b>
       上線，擁有超過兩萬個不同主題的看板，每日有上萬篇的新文章被發表以及閱讀，並且擁有前輩朋友留下數量可觀的資料文件。
       <br/>
       本站並時常與
       <a href="/links.html#edu">
        教育界
       </a>
       、
       <a href="/links.html#org">
        公益界
       </a>
       、
       <a href="/links.html#mu">
        各大媒體
       </a>
       、
       <a href="/links.html#gov">
        政府機關
       </a>
       、以及
    Open Source Software  等等各界的朋友合作，在
       <a href="links.html">
        link
       </a>
       中可以查詢與本站合作朋友的列表；亦同時歡迎各界的朋友與我們合作，詳情請見
       <a href="contact.html">
        與我們聯絡
       </a>
       。
       <small>
        (恕不接受任何政治、商業的合作申請！)
       </small>
      </p>

      <h2>
       批踢踢實業坊支持網站分級
      </h2>
      批踢踢實業坊支持網站分級政策，已經在
      <a href="telnet://ptt.cc">
       批踢踢
      </a>
      、
      <a href="http://ptt2.cc">
       批踢踢兔
      </a>
      的 telnet-based 介面，各板板主可依各板討論內容，自行決定該板是否開放十八歲以下使用者閱讀。一旦設定，將
      <b>
       強制拒絕十八歲以下使用者進入
      </b>
      。同時在
      <a href="/bbs/index.html">
       web bbs
      </a>
      及
      <a href="/man/index.html">
       網頁版精華區
      </a>
      對限制級內容加入機器分級標示。若您尚在使用微軟的 Internet Explorer ，還未更新到更快更好更安全的
      <a href="https://moztw.org">
       FireFox
      </a>
      ，請參閱
      <a href="http://www.ticrf.org.tw/chinese/html/06-rating-3.htm">
       安裝說明
      </a>
      ，或拜訪
      <a href="http://www.ticrf.org.tw/">
       財團法人網站分級推廣基金會
      </a>
      。
      <br/>
      <small>
       last update: Mon May 8 11:10:55 CST 2006
      </small>
