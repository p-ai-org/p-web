     <div class="intro">
      <p>
       Processing is a flexible software sketchbook and a language for learning how to code within the context of the visual arts. Since 2001, Processing has promoted software literacy within the visual arts and visual literacy within technology. There are tens of thousands of students, artists, designers, researchers, and hobbyists who use Processing for learning and prototyping.
      </p>
      <ul>
       <li>
        » Free to download and open source
       </li>
       <li>
        » Interactive programs with 2D, 3D, PDF, or SVG output
       </li>
       <li>
        » OpenGL integration for accelerated 2D and 3D
       </li>
       <li>
        » For GNU/Linux, Mac OS X, Windows, Android, and ARM
       </li>
       <li>
        » Over 100 libraries extend the core software
       </li>
       <li>
        » Well
        <a href="./reference/">
         documented
        </a>
        , with many
        <a href="./books/">
         books
        </a>
        available
       </li>
      </ul>
     </div>
     <div class="intro">
      <p>
       <a class="large-link" href="./books/">
        Books
       </a>
       <br/>
       <br/>
       The second edition of
       <a href="./books/#reasfry2">
        Getting Started with Processing
       </a>
       is here and it's updated for Processing 3. It's now in full color and there's a new chapter on working with data. The
       <a href="./handbook/">
        Processing Handbook
       </a>
       now has a second edition too. Every chapter has been revised, and new chapters introduce more ways to work with data and geometry. New “synthesis” chapters offer discussions and worked examples of topics such as sketching with code, modularity, and algorithms.
       <br/>
       <br/>

       </a>
      </p>
